# essays/archive

Superseded versions of essays, kept intact for later comparison.

Nothing is ever lost regardless: every past revision of every essay lives
permanently in the repository's git history and can be recovered by commit.
This folder simply keeps the most notable prior versions where they are easy
to open and compare side by side with the current live text.

## Contents

- `essay-01-cambrian-horror--v-2026-07-17.md` — the pre-v3 version of essay 01
  ("Das Grauen im Kambrium / The Horror in the Cambrian"), as of 2026-07-17
  (git commit `c872040`). The live essay was rewritten into the v3 voice
  ("Feynman leads, Bernhard punctuates") on 2026-07-19; see
  `editorial/STYLE-essays.md`.

To see the full revision history of any essay:
`git log --follow -- essays/<file>.md`
